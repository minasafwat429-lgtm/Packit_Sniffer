# Packet Sniffer Tool

A simple Python-based packet sniffer built using Scapy.  
This tool captures network packets and analyzes HTTP requests to display visited URLs and detect possible login credentials in the network traffic.

⚠️ This project is for **educational purposes and authorized network testing only**.

---

## Features

- Capture live network packets
- Detect HTTP requests
- Extract visited URLs
- Identify possible login credentials in network traffic
- Lightweight and easy to use

---

## Technologies Used

- Python
- Scapy

---

## Requirements

- Python 3
- Root / Administrator privileges
- Linux environment recommended
